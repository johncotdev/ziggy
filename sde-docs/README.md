## Project Goals

The Ziggy DBL project aims to:

1. **Learn Zig** - Build practical experience with the Zig programming language
2. **Learn Compilers/Interpreters** - Understand lexing, parsing, code generation
3. **Achieve DBL Feature Parity** - Compile and run DBL programs
4. **Build Custom ISAM** - Replace proprietary ISAM with open-source Zig implementation
5. **Break Vendor Lock-in** - Provide an open-source alternative for DBL applications
